### 🛠 Docker Desktop インストール手順

#### **【共通の準備】**

1.  **システム要件の確認**:
    *   **Windows**: Windows 10 64-bit: Pro, Enterprise, Education (バージョン2004以降) または Home (バージョン22H2以降)。WSL 2（Windows Subsystem for Linux 2）の有効化が必要です。
    *   **macOS**: macOS Big Sur 11以降。IntelチップまたはApple Silicon（M1/M2/M3チップ）搭載Mac。
    *   **メモリ**: 最低4GB RAM。

#### **【Windowsの場合】**

1.  **WSL 2 の有効化**:
    *   Docker DesktopはWSL 2を利用します。まだ有効化していない場合は、以下の手順で有効化してください。
        1.  管理者権限でPowerShellまたはコマンドプロンプトを開きます。
        2.  `wsl --install` と入力して実行します。
        3.  PCを再起動します。
        4.  `wsl --set-default-version 2` と入力して実行します。
    *   詳細な手順はMicrosoftの公式ドキュメントを参照してください。
2.  **Docker Desktop のダウンロード**:
    *   Docker公式サイトのダウンロードページにアクセスします: [https://docs.docker.com/desktop/install/windows-install/](https://docs.docker.com/desktop/install/windows-install/)
    *   「Download Docker Desktop for Windows」ボタンをクリックしてインストーラーをダウンロードします。
3.  **インストーラーの実行**:
    *   ダウンロードした `Docker Desktop Installer.exe` をダブルクリックして実行します。
    *   インストーラーの指示に従って進めます。通常はデフォルト設定で問題ありません。
    *   インストール完了後、PCの再起動を求められる場合があります。

#### **【macOSの場合】**

1.  **Docker Desktop のダウンロード**:
    *   Docker公式サイトのダウンロードページにアクセスします: [https://docs.docker.com/desktop/install/mac-install/](https://docs.docker.com/desktop/install/mac-install/)
    *   お使いのMacのチップ（IntelまたはApple Silicon）に合わせて適切なインストーラー（`.dmg` ファイル）をダウンロードします。
2.  **インストーラーの実行**:
    *   ダウンロードした `.dmg` ファイルを開きます。
    *   DockerアイコンをApplicationsフォルダにドラッグ＆ドロップします。
    *   ApplicationsフォルダからDockerを起動します。初回起動時にはセキュリティの許可を求められる場合があります。

---

### **【インストール後の確認】**

インストールが完了したら、Docker Desktopが正常に動作しているか確認しましょう。

1.  Docker Desktopを起動します。
2.  タスクバー（Windows）またはメニューバー（macOS）にDockerのクジラのアイコンが表示され、緑色のランプが点灯していれば正常に起動しています。
3.  ターミナル（Windowsの場合はPowerShell、macOSの場合はTerminal）を開き、`docker run hello-world` と入力して実行します。
    *   「Hello from Docker!」のようなメッセージが表示されれば、Dockerは正常に動作しています。
