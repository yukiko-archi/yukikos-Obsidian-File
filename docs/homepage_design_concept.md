# ホームページデザインコンセプト：宮川えりか研究室を参考に

## 全体的な印象
「優しさ」「温かさ」「親しみやすさ」「清潔感」を大切にし、訪問した方がホッと安心できるようなデザインを目指します。DIYの「楽しいけれど少し不安」という気持ちに寄り添い、「ここなら安心して挑戦できる」と感じてもらえるようにします。

## 1. 色合い（カラーパレット）

-   **メインカラー（背景など）:**
    -   **背景色: `#fff7ef` (非常に淡いピーチベージュ)**
    -   **グラデーション開始色: `#fffcf9` / グラデーション終了色: `#fff0e8` (背景に合わせた柔らかなグラデーション)**

-   **アクセントカラー:**
    -   **赤色: `#ff8484` (情熱や活力を表現する、明るい赤)**
    -   **青色: `#2e6be6` (信頼感や安心感を表現する、鮮やかな青)**

-   **文字色:**
    -   **基本の文字色: `#000000` (黒)**

**補足:**
以前設定されていた薄いグリーン（`--primary-color`）は、ご要望により廃止し、その役割は必要に応じて黒または上記アクセントカラーで代替されています。これにより、赤と青のアクセントがより際立つ、洗練されたデザインとなります。

## 2. イラストの雰囲気とタッチ
宮川研究室のイラストは、ホームページの「顔」とも言える重要な要素です。

*   **タッチ:** 「**手描き風のゆるい線**」と「**フラットで柔らかい塗り**」を組み合わせたスタイル。
*   **特徴:**
    *   **親しみやすさ:** 人物や道具の形は、カチッとしすぎず、少しデフォルメされたり、丸みを帯びたりしていると、見る人に「親しみ」を感じさせます。
    *   **人間味・温かさ:** 線が均一ではなく、少し揺らいだり、手で描いたようなアナログ感が残っていることで、デジタルな画面の中に「人間らしさ」や「温かさ」が生まれます。
    *   **テーマとの融合:** 阿部さんの場合、「DIY」「共創」「安心」「夢を形にする」といったテーマを、日常の一コマを切り取ったようなイラストで表現します。
        *   例：お客様と阿部さんが一緒に設計図を見ているイラスト、楽しそうにペンキを塗っているイラスト、完成した空間で喜ぶ家族のイラストなど。

## 3. レイアウトとタイポグラフィ（文字のデザイン）

*   **余白を大切に:** 宮川研究室のサイトのように、情報が詰まりすぎず、ゆったりとした「余白」を十分に取ります。これにより、上品で落ち着いた印象になり、情報が読みやすくなります。
*   **文字（フォント）:**
    *   **読みやすさ重視:** 本文は、PCやスマホで長時間読んでも疲れない、清潔感のあるゴシック体（日本語フォントなら「Noto Sans JP」などが良いでしょう）を選びます。
    *   **柔らかさ:** 見出しなどには、少し丸みのあるフォントや、字間（文字と文字の間隔）をゆったりと取ることで、全体の優しい雰囲気を損なわないようにします。
